开发系统：win10操作系统
开发工具：eclipse 4.7.0
Java平台：JDK SE 1.8版本

游戏操作说明：

   首页选项=》移动：A；进入：回车（1单人、2双人、3自定义地图）
   1.单人操作：上下左右键+F（发射弹药）
   2.双人操作：甲：WASD+F, 乙：上下左右键+J
   3.自定义模型：先W S键选择关卡地图，按回车确定进入修改，大道具：空格+D键，小道具：A键
	保存：E键，不保存退出：Q键

bug提示：
   1.频繁操作存在随机卡顿的情况，可能是代码优化的不够
   2.关卡跳转是有时候会一下从第一关跳到第三、第四关
   3.单双人模式关卡地图加载标志位共用，所以当你单人模式停留在第四关退出去双人模式会直接从第四关开始     ，退出在进入单人模式也是从第四关开始。
